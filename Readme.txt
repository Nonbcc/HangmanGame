1. เริ่ม run โปรแกรมจะขึ้นหน้าต่างให้กด play game สามารถกดเข้าไปเพื่อไปหน้าต่างถัดไปได้และยังมีปุ่ม Exit เพื่อปิด game ได้ด้วย
2. เมื่อกด play game เข้าหน้าต่างถัดไปจะมีให้เลือกหมวดหมู่เพื่อ link ไปยังหน้าถัดไปมี 3 หมวดหมู่คือ Animals , Foods และ Colours
    แล้วยังมีการเลือก character ทั้งหมด 3 ตัวละครที่จะให้เป็นตัวของเราโดยจะเป็นรูปภาพอยู่ภายใน Folder การที่จะไปหน้าต่าง
    ถัดไปได้จะต้องเลือกหมวดหมู่ 1 อย่างและตัวละคร 1 ตัว ปุ่ม Ok จึงจะขึ้นมาเพื่อให้กดแล้วเข้าสู่หน้าต่างถัดไป
3. เข้าสู่หน้าต่างของการเล่นเกมหลังจากที่กด Ok โปรแกรมจะทำการสุ่มคำจากหมวดหมู่ที่เราเลือกขึ้นมา 1 คำโดยจะมีช่องว่าง
    ให้เติมคำให้สมบูรณ์ โดยจะมีปุ่มกด A ถึง Z โดยเมื่อเริ่มตัวอักษรที่อยู่ภายในคำที่ปรากฏให้เป็นคำใบ้ ปุ่มตัวอักษรจะหาย
    ไปไม่ขึ้นมาให้กดเลือกตัวอักษรหากกดเลือกไปแล้วปุ่มนั้นก็จะหายไป ถ้าเลือกตัวอักษรไปเติมได้ถูกต้องจะมีตัวหนังสือ
    ตัวนั้นขึ้นตรงช่องว่างตามที่โปรแกรมได้ setting เอาไว้แล้ว ถ้าสามาถเลือกได้ถูกจนครบเป็นคำหนึ่งคำจะเด้งไปที่หน้าต่างถัดไป
    ที่เขียนว่า You win แต่ถ้าเลือกตัวอักษรผิดตัวละครที่เราเลือกไว้ตอนแรกก็จะปรากฏขึ้นมาทีละส่วนถูกแบ่งเป็น 6 ส่วน
    แต่สามารถผิดได้ทั้งหมด 7 ครั้ง โดยครั้งที่ 7 จะขึ้นเป็นภาพตัวละครผูกคอพร้อมกับบอกว่า You lose
